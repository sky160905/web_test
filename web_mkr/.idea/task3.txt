Оператор spread  та параметр rest  є інструментами в JavaScript для роботи з масивами та об'єктами.
Оператор spread:

- Він дозволяє розгортати ітерабельні об'єкти (наприклад, масив або рядок) у окремі елементи. Він
використовується для розпакування елементів масиву або властивостей об'єкта.

1. Копіювання масиву:

   const arr1 = [1, 2, 3];
   const arr2 = [...arr1]; // створює копію arr1

2. Конкатенація масивів:

   const arr1 = [1, 2, 3];
   const arr2 = [4, 5, 6];
   const merged = [...arr1, ...arr2]; // об'єднує arr1 та arr2

3. Передача елементів масиву як аргументів у функцію:

   const arr = [1, 2, 3];
   const max = Math.max(...arr); // еквівалентно Math.max(1, 2, 3)

Параметр rest:
Параметр rest дозволяє представити невизначену кількість аргументів у вигляді масиву.

1. Збір аргументів функції:

   function sum(...args) {
       return args.reduce((total, num) => total + num, 0);
   }
   console.log(sum(1, 2, 3)); // результат: 6

2. Збір залишкових елементів масиву:

   const [перший, ...решта] = [1, 2, 3, 4, 5];
   console.log(перший); // результат: 1
   console.log(решта); // результат: [2, 3, 4, 5]


3.Збір залишкових властивостей об'єкта:
   const { ім'я, вік, ...решта } = { ім'я: 'Джон', вік: 30, стать: 'чоловіча', країна: 'США' };
   console.log(ім'я); // результат: 'Джон'
   console.log(вік); // результат: 30
   console.log(решта); // результат: { стать: 'чоловіча', країна: 'США' }

Різниця у використанні полягає в тому, що оператор spread використовується для розпакування елементів
 або властивостей, а параметр залишку використовується для збору елементів або властивостей у один масив чи об'єкт.

Різниця у позиції в коді полягає в тому, що оператор  spread використовується там, де очікується декілька елементів або властивостей.
 Параметр rest використовується у параметрах функцій або деструктуризації для збору залишкових елементів.
